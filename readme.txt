사용하는 법 : main.pyw를 이용해 실행합니다.

각각의 버튼은 다음의 기능을 수행합니다.
Select File : 다중의 파일을 선택합니다.
Delete File : 리스트에 선택된 파일을 리스트에서 제거합니다.
make DB     : DB 파일을 만들기 위한 분석을 수행하고 DB파일을 생성합니다.
DB Open     : DB 파일을 오픈합니다.
Export      : DB 파일을 CSV로 Export합니다.

db 생성 후 메일 각 라인을 더블 클릭하면 해당 메일의 Content html파일과 첨부파일 리스트를 볼 수 있으며
해당 창에서 파일을 더블 클릭하면 실행 됩니다.
